### svn增量更新文件查找
本工具为查找svn修订版本增量更新文件小工具。<br>
前提约定项目目录为maven标准目录，且使用svn做为版本管理工具。<br>
<br>
打包文件名：输入压缩文件的文件名，一般为问题号<br>
基础版本：查看svn提交日志，选择你要导出更新的前一次修订版本做为基础版本<br>
修订版本：查看svn提交日志，选择要导出更新的版本号<br>
项目路径：选择项目在磁盘中的目录，svn托管目录<br>
填好信息后点击打包<br>
文本区域内列出打包文件并统计文件个数<br>
